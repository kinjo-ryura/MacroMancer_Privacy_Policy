Privacy Policy

1. Information Collected

This app collects the following information from your macOS device:

  • Application names
  • Application file paths
  • Application icons

This information is not personally identifiable.

2. Use of Information

Data is used solely for displaying and launching macOS apps from an iOS device.

3. Data Transmission and Storage

All communication takes place locally via Bluetooth. No data is sent to any server. The information is stored only on your device using CoreData.

4. Bluetooth Usage

Bluetooth is used only for communication between your iOS and macOS devices. This app does not access location data.

5. User Control

You can stop data sharing at any time by turning off Bluetooth or uninstalling the app.

6. Contact

If you have any questions, contact us at: macromancer.app@gmail.com
